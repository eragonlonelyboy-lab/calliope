# Phase 4 research pass (generic procedure)

The Strategist runs this; the active medium pack's "research notes" section adds medium-specific stations. Depth is the client's purchase: offer the dial with effort shown before dispatch.

## Depth dial (offer exactly these three)

| Depth | Contents | Rough effort |
|-------|----------|--------------|
| Shallow | Named competitors only, visual-language read, 1-line whitespace note | minutes; no external research beyond the sites themselves |
| Standard (default) | Full competitive audit + visual-whitespace map + scoped trend pass | one research dispatch |
| Deep | Standard + adjacent-market sweep + audience-context research + trend sourcing with citations | multiple dispatches; quote before running |

State the effort in plain terms; if any paid API or heavy crawl is involved, price it (cost-transparency rule).

## Stations (standard depth)

1. **Competitive audit.** The client's named competitors (Phase 1 2×2) + up to 3 the client did not name but the buyer sees. Per competitor: positioning read, visual language (type/color/motion in one line each), what they own in the buyer's mind, one concrete observation worth stealing or avoiding.
2. **Visual-whitespace map.** Overlay the audit on the 2×2: which visual/verbal territory is unoccupied AND reachable from the client's soul answers. Output: 2-3 named territories with "what occupying it would look like" sketches in words. This map feeds treatments directly; without it, treatments are taste guesses.
3. **Trend pass.** Scoped three ways: the medium, the industry, and the soul-interview feeling. Named trends only, each with source + date + adopt/adapt/ignore recommendation. Trends inform; the bar governs; a trend that violates a client principle is listed as "seen and rejected" so the client knows it was considered.

## Output

`.calliope/research/competitive_audit.md` + `trends.md` (formats in `references/roles/strategist.md`). Return summary: the 3 findings that should change what we build. The CD reads both files before treatments; the reveal cites research where it shaped a concept.
